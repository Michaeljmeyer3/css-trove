The Button-Gradient-Border

A Simple Button with a gradient border with pseudo decorations

It utilizes 3 elements.

  • BBG-Container:
      • Containerization
  • BBG-botton-border:
    • A Div with a Liner Gradient for a background.
    • With a little padding, it appears as a border
    • Hover states to help flip the background for a dynamic feel.
  • BBG-Button: HTML Button
